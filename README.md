**Wallet Application**
Учебное backend-приложение для практики разработки на Spring Boot, проектирования REST API и контейнеризации с использованием Docker.
Приложение реализует базовую логику кошелька с возможностью:
- создания кошелька,
- пополнения баланса,
- списания средств,
- получения текущего баланса.

**API**
Базовый путь: /api/v1/wallet

**Создание кошелька**
POST /api/v1/wallet

Пример запроса:
{
"walletId": "UUID"
}

Ответ:
201 Created
Тело ответа содержит UUID созданного кошелька.

**Операция по кошельку**
POST /api/v1/wallet/operation

Пример запроса:
{
"walletId": "UUID",
"operationType": "DEPOSIT",
"amount": 1000
}

Где:
walletId — идентификатор кошелька
operationType — тип операции (DEPOSIT или WITHDRAW)
amount — сумма операции

После выполнения операции возвращается:
{
"walletId": "UUID",
"balance": 1500
}

**Получение баланса**
GET /api/v1/wallet/{walletId}

Возвращает текущий баланс кошелька:
{
"walletId": "UUID",
"balance": 1500
}

**Обработка ошибок**
Реализован глобальный обработчик исключений через @ControllerAdvice.
Корректно обрабатываются:
- попытка операции по несуществующему кошельку,
- недостаточный баланс при списании,
- невалидный JSON,
- ошибки валидации входных данных.
Ответы возвращаются в унифицированном формате.

**Архитектура**
- REST-контроллер
- Сервисный слой с транзакционной логикой
- Репозиторий на основе Spring Data JPA
- PostgreSQL как основная база данных
- H2 для тестирования
- Используется BigDecimal для точных финансовых расчётов
- Retry-механизм отсутствует

**Запуск через Docker**
Сборка Docker-образа
**gradle clean docker**

Запуск контейнеров
**docker compose up**

Приложение будет доступно по адресу:
http://localhost:8080
